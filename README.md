# Exploratory-Data-Analysis-Inference
DES432 Project 1: Exploratory Data Analysis &amp; Inference

## Dataset Description
- **Source**: Udon Thani Hospital Archive (The data was properly requested with acknowledge by hospital chair person)
- **Unit of Analysis**: Individual clinical visits.
- **Key Variables**: 
  - `Age`: Numeric, patient age.
  - `sex`: Categorical (Male/Female).
  - `Occupation`: Categorical, patient's primary work.
  - `ICDCode`: Diagnosis code (G20 for Parkinson's disease group).
